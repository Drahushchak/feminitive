# feminitive

Сайт написаний на Flask фреймворці. Задеплоїний на Heroku (сервер де розміщений застосунок). Використовується база даних PostgreSQL для зберігання даних, яка теж розміщена на Heroku. Для взаємодії між базою даних і застосунком використовується ORM (Object-Relational Mapping - Об'єктно-реляційна проекція) flask_sqlalchemy. flask_sqlalchemy застосовує бібліотеку sqlalchemy і для адекватної взаємоді з PostgreSQL вастановлено ще одну бібліотеку psycopg2, яка транслює запити з sqlalchemy до PostgreSQL. Для міграції даних з ORM на базу даних використовується Flask-Migrate бібліотека. Для взаємодії застосунку і відкритого веб-сервера на Heroku, а також для регулювання кількості потоків використовується Gunicorn WSGI HTTP Server (WSGI - Web Server Gateway Interface). Для генерації формів на сайті застосовується бібліотека WTForms (таких форм як Login і Sign Up). Використана бібліотека PyYAML для зчитування .yaml файлу і перетворення його в Python dict object. Для простішої верстки застосовано Bootstrap4 стилі і JS сценарії. Для побудови діаграм було підключено Chart.js
